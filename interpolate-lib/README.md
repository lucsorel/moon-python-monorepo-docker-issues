# interpolate-lib

`interpolate-lib` is a Python library that provides interpolation utilities and shared logic for the monorepo.

## Installation

Install the package with `uv` or your preferred package manager once the repository is configured:

```bash
uv install
```

## Usage

Import the library from your Python code:

```python
from interpolate_lib import linear_interpolate
```

## Project layout

- `src/interpolate_lib/` – library source package
- `pyproject.toml` – package metadata and dependencies
